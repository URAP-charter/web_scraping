This document describes the Internationalized Domain Names (IDN) Character Table to be used by NIC Chile in the .CL TLD registry for the registration of .cl domain names.

We advise registrants to carefully consider the potential confusion over the perception of equivalence between an accented Latin character and another sequence of Latin characters (with or without accents). There exists no equivalence rule nor character mappings.

This table is in compliance with the ICANN Guidelines for the Implementation of Internationalized Domain Names Version 2.1, and is intended for publication in the IANA IDN Character Table Registry.

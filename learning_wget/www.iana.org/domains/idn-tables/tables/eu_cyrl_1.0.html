This document describes the Cyrillic Internationalized Domain Names (IDN) Character Table to be used by EURid vzw/asbl, the .eu Registry for registration of .eu Internationalized Domain Names in Cyrillic Script.

The hyphen (U+002D), as well as the digits zero to nine (U+0030 .. U+0039), for purposes of .eu Internationalised Domain Name registration are considered to form part of the Cyrillic script.

This table is in compliance with the ICANN Guidelines for the Implementation of Internationalized Domain Names Version 2.2, and is intended for publication in the IANA IDN Character Table Registry.

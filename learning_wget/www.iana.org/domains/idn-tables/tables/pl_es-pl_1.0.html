This document presents a character table created by NASK (Research and Academic Computer Network - .PL ccTLD registry) for Spanish language. The document is intended for informational purpose only and for exclusive use by .PL ccTLD Registry. The character set was created on the basis of the following sources:
1. http://www.evertype.com/alphabets/index.html
2. http://www.omniglot.com
3. http://www-950.ibm.com/software/globalization/icu/demo/locales
The table does not provide neither definition of an alphabet nor language itself. The table may contain additional letters which are used to represent foreign names and words. These letters may not be listed in the alphabet, or may be listed very rarely. Full policy of IDNs registration is presented in the following document "Registering Internationalized Domain Names under .PL" A. Bartosiewicz, K. Olesik (http://www.dns.pl/IDN/idn-registration-policy.txt).

The following characters are given in their Unicode code points in format U+XXXX, where X is a hexadecimal digit. The range of code points is expressed as U+XXXX..U+YYYY, where XXXX and YYYY are the first and last Unicode code point in the range.

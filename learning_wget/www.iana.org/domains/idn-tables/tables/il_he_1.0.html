This document describes the IDN table used by the Israel ccTLD registry, The Israeli Internet Association (ISOC-IL), for the registration of domain names using the Hebrew script. Hebrew is the primary official language in Israel.

This table is presented in compliance with the ICANN "Guidelines for the Implementation of Internationalized Domain Names" Version 2.1.

Characters are listed in this table by their Unicode code points in U+xxxx format, and the Unicode name for each character after a hash sign "#" on the same line.

A Hebrew Domain Name label registered under .il must meet the following requirements:

  • Be comprised of characters from this table, only.
  • Be at least 2 characters long in U-Label form and at most 63 characters long in A-Label form.
  • Not start with, end with, or contain two consecutive occurances of, U+002D (HYPHEN-MINUS).
  • Meet IDNA2008 requirements for bidirectional language labels (RFC5893); in particular, must start with a Hebrew letter and must end with a Hebrew letter or a digit.

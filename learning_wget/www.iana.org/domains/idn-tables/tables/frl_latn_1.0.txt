This document describes the Latin Internationalized Domain Names (IDN) Character Table to be used by FRL Registry B.V. in Latin Script.

The hyphen (U+002D), as well as the digits zero to nine (U+0030 .. U+0039), for purposes of Internationalised Domain Name registration are considered to form part of the Latin script.

This table is in compliance with the ICANN Guidelines for the Implementation of Internationalized Domain Names Version 3.0, and is intended for publication in the IANA IDN Character Table Registry.

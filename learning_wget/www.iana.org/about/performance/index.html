Reporting on Performance

We seek to provide an excellent, reliable and performant service of our various registration roles. To achieve this, we regularly review our procedures and liaise with our user communities to optimise performance.

More formally, performance standards are defined as part of various contracts we have that govern the performance of our services.

Protocol Parameters

Report Description
IETF Statistics Report Documentation of the performance of the protocol assignments roles performed by ICANN for the IETF community. (Monthly)
Internet Draft Processing Status Information on pending Internet Draft actions being evaluated by us.

Domain Names

Report Description
SLE Dashboard Real-time reporting of performance metrics defined by the naming community for root zone management performance.
PTI Naming Performance Report A report of performance metrics produced monthly for the Customer Standing Committee.
Root Zone Audit Data A report of all root zone related changes transacted. (Monthly)
